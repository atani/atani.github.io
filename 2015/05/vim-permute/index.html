vimで置換して改行

05-26-2015

vim でログファイルを開いたときに、本来ならば改行されている箇所が \n\r となっていて、１行にログが出ていて非常にみづらい状態になっていた。

そこで、改行を置換。
改行は Ctrl + v + Ctrl + m をタイプすることで、 ^M となる。

  1. vimで対象ファイルを開く
  2. コマンドモードで :%s/\\n\\r/^M/g + Enter

こうすることで、無事見やすくなった。
動画をご参照にどうぞ。
